🚀 추천 앱 5종 포탈

GitHub Pages에서 동작하는 브라우저 기반 앱 모음입니다.

1) 액션캠 저장 용량 계산기

해상도/프레임/비트레이트 기반으로 예상 촬영 시간을 계산합니다.

2) AI 프롬프트 템플릿

역할/목적/형식을 입력해 프롬프트를 생성하고 저장합니다.

3) 아티스트 타임라인 Fan Hub

연도별 사건과 세계관 노드를 인터랙티브하게 탐색합니다.

4) 한국어 의성어·의태어 퀴즈

플래시카드 형태로 문제를 맞추고 정답을 확인합니다.

5) B2C 앱 사전예약 랜딩

앱 소개, 스크린샷, 이메일 사전등록 폼까지 포함한 랜딩 페이지.
